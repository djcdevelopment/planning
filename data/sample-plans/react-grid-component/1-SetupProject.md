# Setup React Project

Create a new React TypeScript project with the following structure:

1. Initialize with Vite + React + TypeScript template
2. Add dependencies: `@tanstack/react-table` for grid functionality
3. Set up basic project structure:
   - `src/components/` for React components
   - `src/hooks/` for custom hooks
   - `src/types/` for TypeScript types
4. Configure ESLint and Prettier
5. Verify the project builds and runs with `npm run dev`

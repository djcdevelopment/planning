# Implement API Endpoint

Implement the user management API using Express.js:

1. Set up Express with TypeScript
2. Implement all CRUD endpoints from the schema
3. Use in-memory storage (Map) for simplicity
4. Add input validation using Zod
5. Add proper error handling middleware
6. Add request logging
7. Write integration tests using supertest
8. Verify all endpoints work with `npm test`
